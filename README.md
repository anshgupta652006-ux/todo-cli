# ✅ To-Do List CLI

A simple command-line To-Do app built with Python. No external libraries needed.

## 🚀 How to Run

```bash
python todo.py
```

## 💬 Commands

| Command | What it does |
|---|---|
| `add Buy milk` | Adds a new task |
| `list` | Shows all tasks |
| `done 1` | Marks task #1 as done |
| `delete 1` | Deletes task #1 |
| `quit` | Exits the app |

## 📁 Files

| File | Purpose |
|---|---|
| `todo.py` | Main app logic |
| `requirements.txt` | Dependencies (none) |
| `.gitignore` | Files Git ignores |

## 🛠 Built With

- Python 3
- JSON for data storage
